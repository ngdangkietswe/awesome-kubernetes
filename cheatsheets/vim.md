# Vim

## Cursor movement

- `gg`: go to the first line of the document

- `G`: go to the last line of the document


## Edit


## Cut & Paste

- `dd`: delete (cut) a line

- `D`, `d$`: delete (cut) to the end of line

- `dgg`: delete (cut) to the begining of file

- `dG`: delete (cut) to the end of file
